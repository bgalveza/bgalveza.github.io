
# Bruno Galvez — Portfolio (static site)

Multi-page site with a minimal top bar and Space Grotesk font.

## Structure
- `index.html` — Home
- `papers.html` — Papers (cards linking to PDFs/arXiv)
- `about.html` — About
- `misc.html` — Miscellaneous
- `contact.html` — Contact
- `assets/style.css` — Shared styles

## Publish on GitHub Pages
1. Create a new GitHub repository (e.g., `bruno-portfolio`).
2. Upload all files in this folder.
3. In **Settings → Pages**, under **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: `main` (root)
4. Your site will be available at `https://<username>.github.io/bruno-portfolio/`.

## Customize
- Replace image links with your own thumbnails.
- Update email/links in `contact.html`.
- Edit cards in `papers.html` to point to arXiv/DOI/PDF.
